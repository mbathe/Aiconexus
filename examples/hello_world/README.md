# Hello World Example

This is the simplest example of creating an AIConexus agent.

## Running

```bash
poetry run python examples/hello_world/agent.py
```

## What it does

1. Creates a simple agent named "Hello World Agent"
2. Registers a "greet" capability
3. Executes the capability with a sample input
4. Returns a greeting
